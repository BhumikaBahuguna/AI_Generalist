# 🤖 AI Assistant (ChatGPT-lite)

## 📌 Overview

A modular AI assistant built using prompt engineering, prompt chaining, and local LLMs.

---

## 🚀 Features

* Explain concepts
* Generate content (questions, ideas, etc.)
* Summarize text
* Context-aware memory
* Fully offline using Ollama

---

## 🛠️ Tech Stack

* Python
* Requests
* Ollama (Llama 3)

---

## ⚙️ Setup

```bash
pip install -r requirements.txt
ollama run llama3
python main.py
```

---

## 🧪 Example Usage

```text
Explain machine learning
Give me DSA questions
Generate startup ideas
Summarize: AI is transforming industries rapidly
```

---

## 🧠 Architecture

User → Intent Router → Module → Memory → Output
