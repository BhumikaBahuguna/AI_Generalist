# 🤖 FAQ Bot (Local LLM)

## 📌 Overview

This project is a simple AI-powered FAQ bot built using a local LLM via Ollama.
It answers user questions in a structured and concise manner using prompt engineering.

---

## 🚀 Features

* Structured prompt-based responses
* Concise and controlled output
* Runs fully offline using Ollama
* Beginner-friendly implementation

---

## 🛠️ Tech Stack

* Python
* Requests library
* Ollama (local LLM - Llama 3)

---

## ⚙️ Setup Instructions

### 1. Install Dependencies

```bash
pip install -r requirements.txt
```

### 2. Install Ollama

Download from: https://ollama.com

### 3. Run Model

```bash
ollama run llama3
```

### 4. Run the Bot

```bash
python main.py
```
